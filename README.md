# 🏛️ HMIF-Nexus

Sistem Informasi Administrasi & Manajemen Surat **Himpunan Mahasiswa Informatika FT UNISMUH Makassar**.

---

## ✨ Fitur Utama
* **Manajemen Surat:** Kelola surat masuk, surat keluar, dan disposisi.
* **Autentikasi Berbasis Peran:** Akses terpisah untuk Admin dan Sekretaris.
* **Pengelolaan Berkas:** Upload & pratinjau dokumen (PDF, Image, Word).
* **Dashboard Interaktif:** Ringkasan statistik dan grafik aktivitas surat.

---

## 🛠️ Teknologi
* **Backend:** PHP (Native)
* **Database:** MySQL / MariaDB
* **Frontend:** HTML5, CSS3, JavaScript
* **Ikon:** Font Awesome 6
